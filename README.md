### Описание проекта
Этот проект представляет собой веб-сервер на Node.js, построенный с использованием Fastify, TypeScript и Postgres.js.

### Требования

	•	Node.js (версия >= 18.x)
	•	npm (версия >= 6.x)
	•	TypeScript (версия >= 4.x) 
	•	PostgreSQL (версия >= 12.x)
	•	Redis (версия >= 5.x)
